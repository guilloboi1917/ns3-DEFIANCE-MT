# DEFIANCE examples

## Overview

The most important examples of the DEFIANCE module are the following:

| Example | Description |
| --- | --- |
| app-communication-example | shows typical communication pattern between different types of RL apps, uses the CommunicationHelper |
| agent-communication-example | shows how agent apps can exchange messages |
| observation-sharing-example | shows how agent apps can forward observations |
| addition-example | trains mock task with RL |
| handover-example | trains handovers with RL |

## Run examples

To run an example, use the command `ns3 run defiance-<example-name>`.
